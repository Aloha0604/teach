  # 【基础】感兴趣区域ROI

有时候，我们只希望读取读取一张图片的部分区域，我们称之为感兴趣区域ROI（Region Of Interest），以后会经常用到。下面演示显示一幅图像的ROI：

  ```python
"""
Author: Will Wang
Email: WillWang1998@163.com
"""
import cv2

img = cv2.imread('lena.jpg')
roi = img[200:400, 200:400]  # 切片操作 忘记的同学翻前面的Python教程

cv2.imshow('roi', roi)
cv2.imshow('img', img)

cv2.waitKey(0)
cv2.destroyAllWindows()

  ```
